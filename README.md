## Overview
This is a Wallet Simple Demo project was built base RoR and ReactJs.

- ruby: "3.0.1"
- rails: "~> 6.1.1"
- antd: "^4.16.8"

## Getting Started

**1. Clone the source code to your local machine:**

**2. Install dependencies:**

```
bundle install
```

```
rails db:setup
```

```
rails db:migrate
```

```
rails db:seed
```

```
yarn install
```

**5. Start development:**

```
rails c
```
**6. Access the webpage:**
```
http://localhost:3000
```
